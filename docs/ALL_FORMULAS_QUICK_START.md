## üöÄ How to Use (3 Steps)

### Step 1: Import to Google Sheets

1. Open Google Sheets (Google Pro account)
2. File ‚Üí Import ‚Üí Upload
3. Select `AUTO_INSURANCE_ALL_FORMULAS_TEMPLATE.csv`
4. Choose "Replace spreadsheet"

### Step 2: Enter Your Data

In **Row 2**, enter:
- **A2**: City name (e.g., "Miami")
- **B2**: State name (e.g., "Florida")

That's it! All other columns have formulas.

### Step 3: Generate Content

1. **Click on cell C2** (or any cell with =AI() formula)
2. Click **"Generate and Insert"** button
3. Wait for AI to generate content
4. **Fill down Row 2** to create more cities:
   - Select row 2
   - Drag down to row 101 (for 100 cities)
   - Or copy-paste formulas

## ‚ö° Bulk Generation (100+ Cities)

### Method 1: Fill Down

1. Enter 100 cities in A2:A101
2. Enter 100 states in B2:B101
3. Click on C2
4. Select C2:BH2 (all formula columns)
5. Press Ctrl+D (fill down)
6. Click "Generate and Insert" for each =AI() cell
   - Or select range with =AI() cells and bulk generate

### Method 2: Copy-Paste Formulas

1. Complete Row 2 with one city
2. Generate all AI content for Row 2
3. Copy cells C2:BH2
4. Paste to C3:BH101

## üí° Tips

### Before You Start:
- **Enable AI features** in Google Workspace
- Make sure you have **Google Pro** with Gemini access

### Speed Up Generation:
1. Generate Row 2 completely first
2. Copy-paste values (Edit ‚Üí Paste special ‚Üí Values only) to freeze
3. Then copy formulas down for bulk

### If =AI() Shows "Loading..." Too Long:
- Simplify the prompt in that cell
- Try refreshing (Ctrl+Shift+R)
- Check internet connection

### To Freeze Content (Prevent Re-generation):
1. Select all cells with =AI()
2. Copy (Ctrl+C)
3. Edit ‚Üí Paste special ‚Üí Values only
4. This converts formulas to static text
